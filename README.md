ruediPy
=======

Instrument control and data processing for RUEDI gas analyzers

ruediPy is a Python package to control RUEDI instruments and to
load, process, and manipulate RUEDI data.

The Octave/MATLAB implementation lives in the separate repository
https://github.com/brennmat/ruediOct.

ruediPy is distributed as free software under the GNU General Public License
(see LICENSE).

Installation
------------

It is recommended to install ruediPy in a Python virtual environment.

Create and activate a virtual environment:

  python3 -m venv .venv
  source .venv/bin/activate

Install ruediPy directly from GitHub:

  pip install git+https://github.com/brennmat/ruediPy.git

To install a specific release (recommended for reproducibility):

  pip install git+https://github.com/brennmat/ruediPy.git@v0.1.0
